# 🛡️ سورس Hhdo (The Giant) 🛡️

مرحباً بك في سورس **Hhdo**، بوت حماية وتليجرام متطور مبرمج بلغة بايثون (Pyrogram).

---

## 🚀 المميزات (Features)
* 🛡️ **حماية قوية:** حذف تلقائي للروابط والتوجيه.
* ⚡ **سرعة استجابة:** مبني على مكتبة Pyrogram الحديثة.
* 💾 **قاعدة بيانات:** يدعم MongoDB لحفظ البيانات.
* 🛠️ **سهولة التعديل:** مبرمج من الصفر وبشكل مبسط.

---

## 🛠️ التنصيب على السيرفر (VPS)
إذا أردت تشغيل السورس على سيرفرك، فقط نفذ هذه الأوامر:

```bash
git clone [https://github.com/osamavvs/Hhdo.git](https://github.com/osamavvs/Hhdo.git)
cd Hhdo
pip install -r requirements.txt --break-system-packages
python3 main.py
