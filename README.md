# 💎 سورس كرستال | Crystal Source 💎

مرحباً بك في سورس **كرستال**، أقوى وأسرع سورس حماية وإدارة مجموعات مبرمج من الصفر بلغة بايثون.

---

## 🚀 مميزات كرستال (Crystal Features)
* 💎 **شفافية وسرعة:** استجابة فورية للأوامر.
* 🛡️ **حماية فولاذية:** منع الروابط، التوجيه، والسبام.
* ⚙️ **إدارة سهلة:** لوحة تحكم بسيطة للمطور.
* 📊 **قاعدة بيانات:** حفظ الإعدادات عبر MongoDB.

---

## 🛠️ التنصيب (Installation)
```bash
git clone [https://github.com/osamavvs/Hhdo.git](https://github.com/osamavvs/Hhdo.git)
cd Hhdo
pip install -r requirements.txt --break-system-packages
python3 main.py
